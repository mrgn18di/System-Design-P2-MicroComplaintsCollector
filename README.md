Micro Complaints Collector

A simple web app where users can submit small everyday complaints. Built with HTML, CSS, JavaScript, Node.js, Express, and MongoDB Atlas for System Design Project 2.


Features
- Submit a complaint text
- Choose a category
- Select intensity (mild, bad, meltdown)
- Saves data to MongoDB Atlas
- Frontend + backend connected through a POST API route


Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MongoDB Atlas (using Mongoose)


How to Run
1. Install dependencies: npm install
2. Create a .env file with: MONGO_URL=your-mongodb-connection-string
3. Start the server: node server.js
4. Open the app in a browser: http://localhost:3000


Project Structure
- index.html
- style.css
- script.js
- server.js
- .env (not included)
